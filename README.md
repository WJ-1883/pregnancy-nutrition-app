# Pregnancy Nutrition Guidance App

## Overview
A mobile application concept designed to support pregnant women with nutrition education, food logging, allergy awareness, pregnancy weight tracking, and prenatal vitamin reminders.

## Features
- Food logging
- Nutrition tracking
- Allergy awareness
- Pregnancy weight tracking
- Prenatal vitamin reminder
- Personalised food guidance

## Technologies / Tools
- SQL Server
- Android Studios
- Database design
- Figma
- UX/UI design
- Research-based app planning

## My Contribution
- Designed the project concept and scope
- Created the database structure
- Designed food, nutrient, allergy, and user profile tables
- Created UI wireframes/prototype
- Researched pregnancy nutrition requirements
- Documented app features and limitations

## What I Learned
- Database design for healthcare-related applications
- User-centred design
- Nutrition and allergy data modelling
- Scope management in software projects
